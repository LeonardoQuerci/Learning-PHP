# Learning PHP 🐘

Bem-vindo ao meu repositório de estudos de **PHP**. Aqui documento a minha evolução na linguagem, desde os conceitos fundamentais até ao desenvolvimento web moderno.

## 📌 Objetivos
O foco deste repositório é consolidar o conhecimento em:
- Sintaxe básica e tipos de dados.
- Manipulação de arrays e superglobais (`$_GET`, `$_POST`).
- Integração com bases de dados (PDO).
- Programação Orientada a Objetos (POO).

## 🚀 Como Executar
Não precisas de um servidor complexo para testar estes exercícios.
Estou utilizando o XAMPP.
